💳 Sistema Bancário em Python

Sistema bancário desenvolvido em Python para execução no terminal, permitindo gerenciamento de clientes, operações financeiras e histórico de transações.

O projeto foi criado com foco em organização de código, modularização e prática de lógica de programação, simulando funcionalidades básicas de um banco.

==================================================================================================================================
📌 Funcionalidades

✔ Cadastro de clientes
✔ Listagem de clientes cadastrados
✔ Remoção de clientes
✔ Transferência entre contas
✔ Histórico de transações
✔ Registro de data e hora das operações
✔ Interface no terminal utilizando Rich

==================================================================================================================================

🖥 Demonstração do Sistema

Exemplo do menu principal no terminal:

MENU DOS CLIENTES

[1] Cadastrar cliente
